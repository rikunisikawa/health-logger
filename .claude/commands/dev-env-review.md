# /dev-env-review — Claude Code 開発環境レビュー

## 実行手順

### 1. 公式ドキュメントの取得

fetch ツールで以下を確認する（取得できない場合は「参照不可」と明記）。

- https://docs.anthropic.com/en/docs/claude-code/settings
- https://docs.anthropic.com/en/docs/claude-code/hooks
- https://docs.anthropic.com/en/docs/claude-code/mcp
- https://docs.anthropic.com/en/docs/claude-code/memory
- https://docs.anthropic.com/en/docs/claude-code/sub-agents

### 2. リポジトリ設定の調査

以下のファイルをすべて読み込む。

```
CLAUDE.md
.claude/settings.json
.mcp.json
.claude/agents/*.md
.claude/commands/*.md
.claude/hooks/*.py
.claude/rules/**/*.md
.claude/skills/*/SKILL.md
```

### 3. レビュー実施

`dev-env-review` スキルのチェックリスト（A〜I）に沿って評価する。

各項目について：
- ✅ 問題なし
- ⚠️ 軽微な改善余地あり
- ❌ 要修正

### 4. 改善案の書き込み

`docs/claude-code-dev-env-review.md` にスキルで定義したフォーマットで出力する。

**制約**: このファイル以外は変更しない。

### 5. コミット

```bash
git add docs/claude-code-dev-env-review.md
git commit -m "docs: Claude Code 開発環境の改善レポートを更新 ($(date +%Y-%m-%d))

Generated by /dev-env-review command.
Co-Authored-By: Claude Code <noreply@anthropic.com>"
```

## 注意事項

- 既存の実装ファイルを変更しない（レビューのみ）
- 不明な点は「不明」と明記する（推測しない）
- 大規模リファクタリング提案は「今後の拡張案」セクションに限定する
